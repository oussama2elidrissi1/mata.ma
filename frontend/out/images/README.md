# Images du Hero

## Image de fond requise

Pour que l'image de fond s'affiche correctement dans la section Hero, placez une image nommée `hero-background.jpg` dans ce dossier.

### Spécifications recommandées :
- **Format** : JPG, PNG ou WebP
- **Résolution** : 1920x1080 pixels minimum (ou plus pour les écrans haute résolution)
- **Taille** : Optimisée pour le web (< 500KB recommandé)
- **Contenu** : Une image représentant le tourisme au Maroc (monuments, paysages, culture, etc.)

### Nom du fichier
L'image doit être nommée exactement : `hero-background.jpg`

### Note
Si l'image n'est pas présente, un dégradé rouge sera affiché en arrière-plan par défaut.

---

## Logo

Pour afficher le logo dans la section Hero, placez une image nommée `logo.png` dans ce dossier.

### Spécifications recommandées :
- **Format** : PNG (avec transparence de préférence) ou SVG
- **Dimensions** : 200x120 pixels (ou ratio similaire)
- **Taille** : Optimisée pour le web (< 100KB recommandé)
- **Fond** : Transparent de préférence pour un meilleur rendu

### Nom du fichier
Le logo doit être nommé exactement : `logo.png`

### Note
Si le logo n'est pas présent, il ne s'affichera pas (pas d'erreur).
